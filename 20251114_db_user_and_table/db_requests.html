Теория Баз Данных

Терминология

СУБД или БД?

  • СУБД - Система Управления Базами Данных, т.е. программа
  • БД - База Данных, т.е. хранилище

Состав БД и мета-данные

  • Отношение (Relation) - таблица. Таблица состоит из строк и колонок.
  • Кортеж () - строка в таблице. Колонки представляют собой заголовок.

Правило №1: кортежи (строки) и колонки (заголовок) по умолчанию линейно не упорядочены каким-либо способом, который будет сохраняться в дальнейшем. В действительности они организованы в бинарное дерево поиска или ещё какой-либо индес (функцию и структуру поиска).

Одна СУБД может управлять несколькими БД. По умолчанию сразу после установки происходит инициализация системной БД: создаются системные таблицы, пользователи и т.д.

Внутри одной БД таблицы могут быть организованы в схемы. Схемы похожи на папки с файлами.

Для разграничений возможностей предусмотрена система прав пользователей и роли.

В системной БД хранится информация пользователях, правах и других БД, схемах и таблицах.

Пользователи могут иметь возможность входить в систему, создавать БД, схемы и таблицы, работать с данными.

Действия над данными = CRUD

  • C - create - создавать (записывать впервые)
  • R - read - читать (если данные есть или получать ответ, что их нет)
  • U - update - обновлять, изменять, редактировать
  • D - delete - удалять

Правило №2: БД можно редактировать только средствами СУБД - SQL-запросами. Изучите 12 правил Кодда.

Структура запроса

Правило №3: выражения, не заключенные в кавычки нечувствительны к регистру.

Запросы завершаются знаком ;

Ключевые слова:

  • CREATE USER, create user - создать пользователя. Пример: create user wera with password 'queen';
  • CREATE DATABASE Пример: CREATE DATABASE school WITH OWNER wera;
  • CREATE TABLE - CREATE TABLE timetable( dtstart timestamp, dtend timestamp, thesubject varchar(1024) );
  • DROP USER, DATABASE, TABLE - уничтожить пользователя, базу или таблицу
  • INSERT INTO .... VALUES (), ()..... - добавление записей в таблицу
  • SELECT .... FROM ..... [ WHERE ..... ] - выбрать, найти, прочитать
